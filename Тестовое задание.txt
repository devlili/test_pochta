Тестовое задание по Python
 
1.	Прочитать из файла url поисковой системы (яндекс, гугл), перейти на url. В поисковике выполнить поиск по слову «Почта РФ». 
2.	Вывести (print) примерное кол-во найденных результатов. Перейти по ссылке из результата поиска на сайт «https://www.pochta.ru». Ссылку открыть в новой вкладке
3.	Закрыть вкладку с поисковой системой
4.	Нажать кнопку «Войти», использовать для этого js
5.	Нажать (в открывшемся окне) на кнопку «Войти». Проверить, что кнопка неактивна, а поля «Электронная почта или телефон» и «Пароль»  - пустые
6.	Ввести в поле «Электронная почта или телефон» пробелы, затем некорректный номер телефона. Вывести (print) валидационные сообщения под полем для каждого случая
7.	Вернуться на главную страницу, выполнить поиск по сайту, в поле ввести «Совкомбанк»
8.	В результатах поиска проверить, что значение в поле совпадает со значением в информационном блоке с сообщением «Извините, по вашему запросу “….” ничего не найдено…»
9.	Перейти на главную страницу, навести курсор мыши на верхнее меню «Отправить». Выбрать вариант «Посылку»
10.	В разделе «Куда» перейти по каждой из кнопок («По России», «В другую страну» … и т.д.) и создать словарь, где ключом будет название кнопки, а значением плейсхолдер поля под кнопками
11.	Вывести получившийся словарь
12.	Закрыть браузер
 
Создать тестовый проект, разбить сценарий, описанный выше на тесты, организовать их в test-suite. Использовать POM, использовать pytest-фикстуры. Используемый браузер - chrome
